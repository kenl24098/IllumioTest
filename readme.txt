test.py - souce code for parsing the log
input.log - input file for the flow log data to parse
tags.csv - input file for the tag lookup table
output.txt - output file generated by test.py
protocal_numbers.csv - used by test.py to convert protocal number to protocal name. The content is copied from https://www.iana.org/assignments/protocol-numbers/protocol-numbers.xhtml 

To run (python 3.5.4 is used during development):
python test.py
